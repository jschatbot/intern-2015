----
【(1) 形態素解析テストツール：jema.rb】
　《使い方》
　　ruby jema.rb 入力ファイル
　《結果の見方》
　　入力ファイルに対し、形態素解析結果が形態素ごとに出力される。
　　※1文のくくりは「[」〜「]」。
　　※surface 　　　：　入力表記
　　　norm_surface　：　正規化後表記
　　　pos 　　　　　：　品詞

----
【(2) 加工ルールテストツール：rewrite.rb】
　《使い方》
　　ruby rewrite.rb ルールファイル 入力ファイル
　《結果の見方》
　　入力ファイルに対し、加工ルール適用結果が行ごとに出力される。
　　例）入力ファイル
　　　　　A
　　　　　B
　　　　　C
　　　　出力ファイル
　　　　　A'
　　　　　B'
　　　　　C'

----
【(3) シナリオテストツール　：trigger.rb】
　《使い方》
　　ruby trigger.rb ルールファイル 入力ファイル
　《結果の見方》
　　入力ファイルに対し、シナリオ適用結果が「----\n」区切りで出力される。
　　例）入力ファイル
　　　　　A
　　　　　B
　　　　　C
　　　　出力ファイル
　　　　　A'
　　　　　----
　　　　　----
　　　　　C'
　　　　　----
　　　　※シナリオが適用されなかった場合（上記の例ではB）、その行は
　　　　　区切り「----\n」のみが出力される。
